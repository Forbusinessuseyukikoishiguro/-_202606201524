# 🐰 自作ツール紹介：うさうさプロンプター v1.6｜研修・動画台本の読み上げサポート

研修動画やオンライン説明会で「台本を読みながら話すと、どうしても画面の文字を目で追っているのが分かってしまう」「カメラ目線を保ちながら自然に話したい」という悩みはありませんか。

今回は、そんな場面のために自作した単一HTMLファイルのツール「**うさうさプロンプター**」をご紹介します。

## どんなツールか

台本（Markdown風のテキスト）を流し込むと、設定した速度で自動的に文字が流れていく、いわゆる「プロンプター（カンペ）」アプリです。ニュースキャスターが原稿を読むときに使う機材のソフトウェア版、とイメージしていただくと分かりやすいかと思います。

- 単一ファイル・オフラインファースト・外部依存なし（vanilla JS）
- ブラウザだけで動作（インストール不要）
- 入力したデータはブラウザ内に自動保存

## 研修講師として使える場面

- 研修動画・eラーニング素材のナレーション読み上げ
- オープニング・クロージングの定型あいさつの読み上げ練習
- オンライン説明会・ウェビナーでの台本読み上げ
- 新入社員研修の自己紹介・発表練習（一定速度で話す訓練として）

## 主な機能

**読み上げ表示**
- 流れる向き：横書き（縦スクロール）／縦書き（右→左・左→右）に対応
- 読み上げ速度を「字/分」で指定し、自分のペースに合わせて調整可能
- 文字色・背景色のカスタム配色（コントラスト比の自動チェックつき）
- 画面ミラー表示（対面プロンプター機材での利用も想定）

**台本管理・再開のしやすさ**
- 中断した箇所から再開／しおり機能／最終ページから、など開始位置を細かく選べる
- 重要な行へのマーカー（ハイライト）づけが可能
- ページ区切り（改ページ）にも対応し、長尺の台本を分割管理できる

**進行サポート**
- タイマー機能（カウントダウン＋アラーム音）
- 操作音（クリック音／ベル音）のON/OFF
- 画面スリープ防止（Wake Lock）で、長時間の読み上げ中も画面が暗くならない
- 読み上げ実績ログ（予定時間×実際の読み上げ時間の記録）

**その他**
- メモ＆ノート機能、JSON/テキストでのエクスポート・インポート
- バージョン履歴の確認
- 「うさうさアナウンサー」によるAI相談機能（台本の読みやすさや話し方の相談、要インターネット接続）

## 使ってみた所感、大歓迎です

速度を「字/分」という具体的な数値で管理できるのが地味に便利で、「このパートは少し速め」「ここは聞き手に考えてもらう時間を作るためゆっくり」といった調整を、感覚ではなく数値で再現できます。研修動画の録り直しが減り、結果的に作業時間の短縮につながりました。

改善要望や不具合報告も歓迎です（ツール内にバグ報告機能を用意しています）。
